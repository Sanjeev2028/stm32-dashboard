A premium, glassmorphism-styled dashboard for the STM32L4 Discovery IoT Node.

## Features
*   **Real-time Monitoring**: Temperature, Humidity, and Pressure.
*   **Live Charts**: Visual trends using Chart.js.
*   **Voice Control**: "Read Sensors" command support.
*   **PWA Support**: Installable on Android/iOS as a native app.
*   **MQTT**: Communicates directly with Adafruit IO.

## How to Use
1.  Open the hosted URL.
2.  Add to Home Screen (Mobile) for the full app experience.
